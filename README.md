# Portfolio Project

Multilingual personal website with CV integration.

## Tech Stack
- Next.js App Router
- TypeScript
- Tailwind CSS
- next-intl (i18n)
- next-seo
- React Hook Form + Zod

## Supported Languages
- English (en) - default
- Turkish (tr)
- German (de)

## Setup
```bash
npm install
npm run dev
```

## Build
```bash
npm run build
npm run start
```

## Sitemap
```bash
npm run sitemap
```
